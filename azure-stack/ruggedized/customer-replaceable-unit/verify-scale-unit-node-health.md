# <a name="verifying-scale-unit-node-health"></a>正在验证缩放单位节点的运行状况

1.  在 **管理门户** 中，验证是否按预期为节点填充了 " **核心** " 和 " **内存** " 列。
    
    如果看到虚线，请继续等待，直到显示正确的数字。 如果一小时过后，短划线仍存在，请联系 Microsoft 支持部门以获得帮助。
    
2.  在 " **区域管理**" 的 " **常规** " 选项卡下，选择 " **警报** "，并确认没有新的警报。
