# <a name="powering-off-the-hardware-lifecycle-host"></a>关闭硬件生命周期主机

验证其运行状况后，可以关闭硬件生命周期主机 (HLH) 。 使用以下过程关闭 HLH，以便可以替换硬件组件。

**步骤**

1.  登录到 HLH。

2.  选择 " **开始" 按钮** ，然后选择 " **关闭** " 以关闭 HLH。

    ![屏幕截图，其中显示了选定的 Windows 按钮和 "关闭"。](media/image-22.png)

    等待几分钟，让关闭完成。
