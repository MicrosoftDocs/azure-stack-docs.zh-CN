# <a name="get-akshcieventlog"></a>Get-AksHciEventLog

## <a name="synopsis"></a>摘要
获取 Azure Stack HCI PowerShell 模块上的 Azure Kubernetes 服务中的所有事件日志。

## <a name="syntax"></a>语法

```powershell
Get-AksHciEventLog
```

## <a name="description"></a>说明
获取 Azure Stack HCI PowerShell 模块上的 Azure Kubernetes 服务中的所有事件日志。

## <a name="examples"></a>示例

### <a name="example"></a>示例
```powershell
PS C:\> Get-AksHciEventLog
```
