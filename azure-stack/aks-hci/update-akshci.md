# <a name="update-akshci"></a>Update-AksHci

## <a name="synopsis"></a>摘要
将 Azure Kubernetes 服务主机更新到最新的 Kubernetes 版本。

## <a name="syntax"></a>语法

```powershell
Update-AksHci
```

## <a name="description"></a>说明
将 Azure Kubernetes 服务主机更新到最新的 Kubernetes 版本。

## <a name="examples"></a>示例

### <a name="example"></a>示例
```powershell
PS C:\> Update-AksHci
```
