# <a name="install-akshci"></a>Install-AksHci

## <a name="synopsis"></a>摘要
Azure Stack HCI 代理/服务和主机上安装 Azure Kubernetes 服务。

## <a name="syntax"></a>语法

```powershell
Install-AksHci
```

## <a name="description"></a>说明
Azure Stack HCI 代理/服务和主机上安装 Azure Kubernetes 服务。

## <a name="examples"></a>示例

### <a name="example"></a>示例
```powershell
PS C:\> Install-AksHci
```
