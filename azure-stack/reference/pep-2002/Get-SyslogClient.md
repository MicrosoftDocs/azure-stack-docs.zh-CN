# <a name="get-syslogclient"></a>SyslogClient

## <a name="synopsis"></a>摘要
获取 syslog 客户端设置。

## <a name="syntax"></a>语法

```
Get-SyslogClient [-AsJob]
```

## <a name="examples"></a>示例

### <a name="example-1"></a>示例 1
```
Get-SyslogClient
```
