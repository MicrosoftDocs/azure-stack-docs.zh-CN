# <a name="get-syslogserver"></a>Get-SyslogServer

## <a name="synopsis"></a>摘要
获取 syslog 服务器终结点。

## <a name="syntax"></a>语法

```
Get-SyslogServer [-AsJob]
```

## <a name="examples"></a>示例

### <a name="example-1"></a>示例 1
```
Get-SyslogServer
```

## <a name="parameters"></a>parameters
